# 油揚げjumpgame
# バグなし　現在修正外にてダウンロードして起動すると動きます
油揚げを天へ導こう<br>
説明<br>
ゲームの簡単な説明：油揚げが上へ進むシンプルゲームで、<br>
リフトに乗りジャンプして高く上りスコアを伸ばすこと目指す。<br>
目標：どこまで上に進めるかを競う。<br>
操作<br>
space/z/↑/上向きボタン:ジャンプ<br>
r/リスタートボタン:リスタート<br>
m/一時停止ボタン:一時停止<br>
n/再開ボタン:再開<br>
a/←/左向きボタン:左へ移動<br>
d/→/右向きボタン:右へ移動<br>
ルール<br>
キー、ボタン操作で移動やジャンプをしリフトの上に乗り<br>
上へ上へと目指していくゲームです<br>
一定以上上へ行くと下のリフトが消えていきます<br>
1000区切りで報酬サウンド、10000区切りで壮大な報酬サウンドがあります<br>
ゲームオーバー条件は下の画面外へ落ちるとゲームオーバーです<br>
